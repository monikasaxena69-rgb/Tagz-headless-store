# Zuno Headless Store

This is a Next.js + Tailwind CSS headless storefront for Zuno.

## Setup

1. Copy `.env.example` to `.env.local` and fill in your Shopify
   store domain and storefront access token.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
